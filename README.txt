Exploring open-ai API here

abstract_summarizer.py: python script which accepts a png screenshot of a journal abstract, parses it, summarizes it with GPT3 in 2 sentences, and prints the result to terminal.

open_ai_embeddings_demo.ipynb: Replicating the embedding demo openai uses. 1. Embeds amazon reviews using GPT3 embedding model. 2. Performs TSNE 2-d visualization of the embedding vectors. 3. Performs random forest regression, attempting to predict rating score from the embedding vector of the review
